# 测试方案文档

## 一、测试概述

### 1.1 测试目标

确保压测系统各功能模块的正确性、稳定性和性能，满足需求规格说明书中的所有功能需求和非功能需求。

### 1.2 测试范围

| 测试类型 | 测试内容 |
|----------|----------|
| 单元测试 | 后端业务逻辑、前端组件 |
| 集成测试 | API 接口、数据交互 |
| 端到端测试 | 用户操作流程 |
| 性能测试 | 系统性能指标 |
| 安全测试 | 用户认证、数据安全 |

### 1.3 测试环境

| 环境 | 数据库 | 服务器 | 说明 |
|------|--------|--------|------|
| 开发环境 | SQLite | 本地 Django | 开发调试 |
| 测试环境 | MySQL | 测试服务器 | 功能测试 |
| 预发布环境 | MySQL | 预发布服务器 | 集成测试 |

from django.test import TestCase
from har_convert.serializers import ScenarioSerializer, RequestSerializer
from har_convert.models import Scenario, Request

class ScenarioSerializerTest(TestCase):
    
    def setUp(self):
        self.scenario_data = {
            'name': '测试场景',
            'description': '测试描述',
            'target_host': 'https://api.example.com',
            'config': {'user_count': 100}
        }
    
    def test_serializer_valid_data(self):
        serializer = ScenarioSerializer(data=self.scenario_data)
        self.assertTrue(serializer.is_valid())
    
    def test_serializer_invalid_data(self):
        invalid_data = self.scenario_data.copy()
        invalid_data['target_host'] = 'invalid-url'
        serializer = ScenarioSerializer(data=invalid_data)
        self.assertFalse(serializer.is_valid())
        self.assertIn('target_host', serializer.errors)
    
    def test_serializer_create_scenario(self):
        serializer = ScenarioSerializer(data=self.scenario_data)
        if serializer.is_valid():
            scenario = serializer.save()
            self.assertEqual(scenario.name, '测试场景')
            self.assertEqual(scenario.target_host, 'https://api.example.com')
