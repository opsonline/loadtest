# 前端页面测试指南

## 测试环境
- 前端地址: http://localhost:3000/demo.html
- 诊断页面: http://localhost:3000/test.html

## 手动测试步骤

### 1. 打开浏览器开发者工具
1. 访问 http://localhost:3000/demo.html
2. 按 `F12` 打开开发者工具
3. 切换到 `Console` (控制台) 标签

### 2. 检查 Console 错误
在控制台查看是否有红色错误信息：
- ❌ 红色 = JavaScript 错误
- ⚠️ 黄色 = 警告（可以忽略）

### 3. 测试登录功能
1. 页面应该显示登录表单
2. 输入用户名: `admin`
3. 输入密码: `admin123`
4. 点击"登录"按钮
5. **预期结果**: 页面切换到主界面

### 4. 测试菜单切换
1. 登录后应该看到左侧菜单
2. 依次点击:
   - 仪表盘
   - 场景管理
   - 报告管理
   - 环境变量
   - 数据源
   - 接口测试
3. **预期结果**: 右侧内容区域切换显示不同页面

### 5. 测试退出登录
1. 点击右上角用户名
2. 选择"退出登录"
3. **预期结果**: 返回登录页面

## 常见问题

### 问题 1: 页面空白
**可能原因**: Vue 或 Element Plus 未加载
**检查方法**: 
- 查看 Network 标签，确认 CDN 资源是否加载成功
- 检查 Console 是否有 "Vue is not defined" 错误

### 问题 2: 点击登录无反应
**可能原因**: JavaScript 错误阻止事件处理
**检查方法**:
- 查看 Console 错误
- 检查是否有 "Cannot read property" 错误

### 问题 3: 菜单无法切换
**可能原因**: 组件切换逻辑错误
**检查方法**:
- 点击菜单时查看 Console 是否有错误
- 检查 `handleSelect` 函数是否被调用
