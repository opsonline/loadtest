# 数据库设计文档

## 一、数据库概述

### 1.1 数据库选型

| 环境 | 数据库 | 说明 |
|------|--------|------|
| 开发环境 | SQLite | 轻量级，无需额外配置 |
| 生产环境 | MySQL 8.0+ | 高并发、可靠性 |

### 1.2 设计规范

| 规范 | 说明 |
|------|------|
| 表命名 | 小写下划线命名，如 `user_profile` |
| 字段命名 | 小写下划线命名，如 `created_at` |
| 主键 | 使用 UUID 作为主键 |
| 时间字段 | UTC 时间，存储格式 `DateTimeField` |
| 软删除 | 使用 `is_deleted` 字段标记删除 |
| 索引 | 高频查询字段添加索引 |

### 6.2 数据迁移脚本

```python
# scripts/migrate_from_json.py
import json
import uuid
from pathlib import Path
from har_convert.models import Scenario, Request

def migrate_scenarios():
    scenarios_dir = Path('scenarios')
    
    for json_file in scenarios_dir.glob('*.json'):
        with open(json_file, 'r', encoding='utf-8') as f:
            data = json.load(f)
        
        scenario = Scenario.objects.create(
            id=json_file.stem,
            name=data.get('name', json_file.stem),
            target_host=data.get('target_host', ''),
            config={}
        )
        
        for idx, req in enumerate(data.get('requests', [])):
            Request.objects.create(
                scenario=scenario,
                name=req.get('url', f'请求{idx}'),
                method=req.get('method', 'GET'),
                url=req.get('url', ''),
                headers=req.get('headers', {}),
                body=req.get('body'),
                body_type='json' if req.get('body') else 'none',
                weight=1,
                think_time=req.get('think_time', 0),
                timeout=30,
                order=idx
            )
```

## 七、备份策略

### 7.1 备份方式

| 备份类型 | 频率 | 保留时间 | 说明 |
|----------|------|----------|------|
| 全量备份 | 每天 | 7 天 | 每日凌晨 3 点 |
| 增量备份 | 每小时 | 24 小时 | 每小时一次 |
| 事务日志 | 实时 | 7 天 | 连续备份 |

### 7.2 备份脚本

```bash
#!/bin/bash
# scripts/backup.sh

BACKUP_DIR="/backup/loadtest"
DATE=$(date +%Y%m%d_%H%M%S)

# MySQL 备份
mysqldump -u root -p${DB_PASSWORD} loadtest | gzip > ${BACKUP_DIR}/loadtest_${DATE}.sql.gz

# 清理旧备份
find ${BACKUP_DIR} -name "*.sql.gz" -mtime +7 -delete

# 上传到对象存储（可选）
aws s3 cp ${BACKUP_DIR}/loadtest_${DATE}.sql.gz s3://backup-loadtest/
```
