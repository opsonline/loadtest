## 📊 项目概况

基于 Web 的压测管理系统，支持 HAR 文件导入、场景管理、压测执行、报告分析等功能。

### 技术栈
- **后端**: Django 4.2 + DRF + Locust + Celery
- **前端**: Vue 3 + Element Plus + Vite
- **数据库**: SQLite (开发) / MySQL (生产)
