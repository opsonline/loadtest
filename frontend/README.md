# 压测系统前端配置

## 环境要求

- Node.js 16+
- npm 或 pnpm

## 启动步骤

1. 安装依赖：
```bash
npm install
# 或
pnpm install
```

2. 启动开发服务器：
```bash
npm run dev
# 或
pnpm dev
```

3. 访问 http://localhost:3000

## 构建

```bash
npm run build
```

## 目录结构

```
src/
├── api/           # API 接口
├── components/    # 公共组件
├── router/        # 路由配置
├── store/         # 状态管理
├── utils/         # 工具函数
└── views/         # 页面视图
```
